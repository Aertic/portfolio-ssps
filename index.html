All projects are deployed using a custom Docker container on my home server running Ubuntu Linux, that I built and secured myself. I came to this approach after Oracle randomly deleted my OCI account with my free tier instances and database, hereby losing all of my data (except local code ofc). So I decided to pick out some parts and a get parts for a semi-powerful homelab server, that I built. My server is protected by cloudflare and an AWS proxy server that's in front of my server. I also have it isolated on my home network for security and have a strict firewall in place. For safety reasons regarding the stability of my internet, I have uploaded all of my websites to an AWS server so they are viewable incase of an outage at my home. These are the default links in the "projects" section. The original links are: docs.polarlab.app/polaris, polarlab.app (/status, /polarmc, /polaris), api.polarlab.app . Všechny projekty jsou spuštěny za pomoci vlastních Docker containerů na domácím serveru, na kterém běží Ubuntu Linux (CLI). Server jsem si sám sestavil a zabezpečil. K tomuto řešení jsem dospěl poté, co mi byl bezdůvodně smazán účet na Oracle Cloud Free Tier, se všemy servery a datbázemi. Rozhodl jsem se tedy vybrat si nějaké díly a sestavit si server, na kterém nyní běží všechny moje projekty a jejich potřebné programy (Databáze atd.). S ohledem na stabilitu našeho internetu jsou pro jistotu všechny stránky přehrané na AWS serveru (odkazy v sekci projekty). Originální odkazy na můj domácí server jsou: docs.polarlab.app/polaris, polarlab.app (/status, /polarmc, /polaris), api.polarlab.app .
